# Advanced Examples

This directory contains comprehensive, real-world examples demonstrating Fowl's capabilities.

## Examples Overview

### 1. Time Series Forecasting
`TimeSeriesForecasting.fsx` - Predict stock prices using LSTM networks

### 2. Image Classification
`ImageClassification.fsx` - CNN for CIFAR-10 classification

### 3. Natural Language Processing
`TextClassification.fsx` - Sentiment analysis with embeddings

### 4. Reinforcement Learning
`Qlearning.fsx` - Q-learning for CartPole

### 5. Generative Models
`VAE.fsx` - Variational Autoencoder for MNIST

### 6. Recommender Systems
`CollaborativeFiltering.fsx` - Matrix factorization for movie recommendations

### 7. Anomaly Detection
`AnomalyDetection.fsx` - Isolation forest for fraud detection

### 8. Portfolio Optimization
`PortfolioOptimization.fsx` - Mean-variance optimization

---

## Running Examples

Each example can be run with:

```bash
dotnet fsi ExampleName.fsx
```

Or compiled:

```bash
dotnet build ExampleName.fsproj
```
